# Admin Dashboard – ASP.NET MVC

## نظرة عامة

هذا المشروع عبارة عن لوحة تحكم إدارية (Admin Dashboard) مبنية باستخدام ASP.NET MVC، وتعتمد على (Three-Tier Architecture). يهدف المشروع إلى توفير بنية منظمة لإدارة البيانات وتنفيذ عمليات CRUD عبر واجهة إدارية واضحة.


## بنية المشروع

يتكون المشروع من :.

### Dashboard.PL (Presentation Layer)

### Dashboard.BLL (Business Logic Layer)

### Dashboard.DAL (Data Access Layer)

## المتطلبات

لتشغيل المشروع تحتاج إلى:
- Visual Studio
- .NET Framework أو ASP.NET MVC 
- SQL Server


## تشغيل المشروع

1. افتح ملف الحل `AdminDashboard.sln` باستخدام Visual Studio.
2. اضبط مشروع `Dashboard.PL` كنقطة تشغيل (Startup Project).
3. شغّل المشروع باستخدام IIS Express أو التشغيل الداخلي من Visual Studio.
